Exploring Tabular Datasets

A pretty common task in data science is exploratory data analysis. Let’s do that. We’ll use the breast cancer dataset from scikit-learn.

from sklearn import datasets
import pandas as pd

def load_cancer_dataset():
    dataset = datasets.load_breast_cancer()
    dataframe = pd.DataFrame(dataset["data"])
    dataframe.columns = dataset["feature_names"]
    dataframe["status"] = dataset.target
    dataframe["status"] = dataframe["status"].replace({0: "malignant", 1: "benign"})

    return dataframe

cancer = load_cancer_dataset()

Back and Forth Chatting

The Explorer, under the hood, has a memory buffer which we can use to have more natural back and forth chats.

Without the buffer, if we asked what is the minimum value of that column? after our initial prompt, it wouldn’t know to what that column refers. Now, since it has a memory buffer, it knows we are talking about the column with the highest mean.
